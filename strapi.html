1
Admin logs into Strapi dashboardt = 0s

Admin navigates to the Strapi admin panel (e.g. admin.yourdomain.com). Logs in via email/password. Super admin sees all content; site editors only see entries where the site relation matches their assigned site — enforced by Strapi RBAC.

Strapi RBACJWT sessionRole-based filter
2
Navigate to ContentBlock for target sitet = 3s

Admin filters ContentBlock entries by site slug (e.g. "madrid"). Strapi's list view shows only blocks belonging to Tejados Madrid. Admin opens an entry — e.g. the hero section title.

Content Type: ContentBlockFilter: site.slug = madrid
3
Edit content, upload media, change SEOt = 10–90s

Admin edits text in the rich text editor, uploads new images to Strapi Media Library (which pushes to Cloudinary/S3), updates SEO fields in the SiteSeo entry. Theme changes (colors, fonts) are updated in SiteConfig.

Rich text editorMedia Library → CloudinarySiteConfig, SiteSeo
4
Click "Publish" → webhook firest = +0.2s

Admin clicks Publish. Strapi saves the entry to PostgreSQL, then fires all registered webhooks for the entry.publish event. The payload includes the content type, entry ID, and site relation data. Strapi signs the request with a shared secret.

entry.publish lifecycleWebhook → signed POSTPostgreSQL persisted
5
Vercel Deploy Hook triggers Astro rebuildt = +2s

A lightweight Vercel Serverless Function receives the webhook, verifies the signature, determines which site(s) to rebuild based on the payload's site slug, and calls the corresponding Vercel Deploy Hook URL(s). Only the affected site rebuilds — not all sites.

Webhook receiver functionSignature verifySelective rebuild
6
Astro builds → fetches Strapi API → deployst = +15–30s

Vercel runs astro build for the target site. During build, Astro fetches all content from Strapi REST API using the site's read-only API token. Pages are generated as optimized static HTML. Vercel deploys to CDN — visitors see the new content.

astro buildGET /api/content-blocks?site.slug=madridStatic HTML → CDN ✅
// 04 — code reference

Key Code Snippets

🚀 Astro — Fetch from Strapi at Build Time
// src/lib/strapi.js — Strapi API client const STRAPI_URL = import.meta.env.STRAPI_URL // e.g. https://cms.yourdomain.com const STRAPI_TOKEN = import.meta.env.STRAPI_API_TOKEN // read-only token for this site const SITE_SLUG = import.meta.env.PUBLIC_SITE_SLUG // e.g. "madrid" async function strapiGet(path) { const res = await fetch(`${STRAPI_URL}/api/${path}`, { headers: { 'Authorization': `Bearer ${STRAPI_TOKEN}` } }) return (await res.json()).data } // Fetch everything for this site in one go export async function getSiteData() { const [config, seo, blocks] = await Promise.all([ strapiGet(`site-configs?filters[site][slug][$eq]=${SITE_SLUG}&populate=logo`), strapiGet(`site-seos?filters[site][slug][$eq]=${SITE_SLUG}`), strapiGet(`content-blocks?filters[site][slug][$eq]=${SITE_SLUG}&sort=sort_order&populate=value_image`), ]) return { config: config[0], seo, blocks } }
🌐 Astro — Page with Strapi Data
// src/pages/index.astro --- import { getSiteData } from '../lib/strapi.js' import Base from '../layouts/Base.astro' import Hero from '../components/Hero.jsx' // Runs at BUILD TIME — fetches from Strapi const { config, seo, blocks } = await getSiteData() const homeSeo = seo.find(s => s.page_slug === '/') const heroBlocks = blocks.filter(b => b.page_slug === '/' && b.is_visible) --- <Base config={config} seo={homeSeo}> <Hero title={heroBlocks.find(b => b.block_key === 'hero_title')?.value_text} image={heroBlocks.find(b => b.block_key === 'hero_image')?.value_image} primaryColor={config.primary_color} /> </Base>
🪝 Webhook Receiver — Vercel Serverless Function
// api/strapi-webhook.js — Vercel serverless function import { createHmac } from 'crypto' // Map site slugs → Vercel deploy hook URLs const DEPLOY_HOOKS = { 'madrid': process.env.VERCEL_HOOK_MADRID, 'barca': process.env.VERCEL_HOOK_BARCA, 'valencia': process.env.VERCEL_HOOK_VALENCIA, } export default async function handler(req, res) { // 1. Verify Strapi signature const sig = req.headers['x-strapi-signature'] const body = JSON.stringify(req.body) const hash = createHmac('sha256', process.env.WEBHOOK_SECRET).update(body).digest('hex') if (hash !== sig) return res.status(401).end() // 2. Determine which site(s) to rebuild const siteSlug = req.body?.entry?.site?.slug const hookUrl = DEPLOY_HOOKS[siteSlug] if (!hookUrl) return res.status(200).json({ skipped: true }) // 3. Trigger rebuild only for the affected site await fetch(hookUrl, { method: 'POST' }) res.json({ rebuilt: siteSlug, timestamp: new Date().toISOString() }) }
